# How to Install the REM Network Chrome Miner

## For Users (Easy Installation)

### Method 1: Load Unpacked (Recommended)

1. **Download the Extension**
   - Download the latest release as a ZIP file
   - Extract it to a permanent location (e.g., `Documents/REM-Miner`)
   - **Important**: Don't delete this folder - Chrome needs it!

2. **Install in Chrome**
   - Open Chrome
   - Go to `chrome://extensions/`
   - Enable "Developer mode" (toggle in top-right corner)
   - Click "Load unpacked"
   - Select the extracted `chrome-miner` folder
   - The REM Miner icon will appear in your toolbar!

3. **Setup Your Miner**
   - Click the extension icon
   - Click "Settings" to open setup wizard
   - Configure RAM commitment (recommended: 4-8 GB)
   - Generate or import a Sui wallet
   - Save your private key somewhere safe!
   - Start mining!

### Method 2: From GitHub

```bash
# Clone the repository
git clone https://github.com/RemNetwork/chrome-miner.git

# Then follow steps 2-3 from Method 1
```

## Troubleshooting

### Extension won't load
- Make sure you extracted the ZIP file
- Don't delete the folder after loading
- Enable "Developer mode" in `chrome://extensions/`

### Connection errors
- Check your internet connection
- Verify firewall isn't blocking WebSocket connections
- Try restarting the extension

### Not earning rewards
- Make sure you're connected (green status)
- Verify your Sui wallet address is correct
- Check uptime is above 80%
- Allow at least 1 hour for first rewards
