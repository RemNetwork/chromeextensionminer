# Quick Start Guide - REM Network Chrome Miner

## Installation (2 minutes)

### Step 1: Load Extension
1. Open Chrome browser
2. Go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked" button
5. Navigate to and select the `chrome-miner` folder
6. ‚úÖ Extension installed!

### Step 2: Configure Mining
1. Click the REM Miner icon in your browser toolbar
2. Click "Settings" button in the popup
3. **Setup Wizard will open:**

#### Configuration (Step 1/3)
- **RAM Commitment**: Use the slider to choose 1-32 GB
  - More RAM = Higher rewards
  - Recommended: 4-8 GB for desktops
  - Start with 2 GB if unsure
- **Referral Code** (optional): Leave default or enter a friend's code
- Click **Continue ‚Üí**

#### Wallet Setup (Step 2/3)
Choose one option:

**Option A: Create New Wallet** (Recommended)
1. Select "Create New Wallet"
2. Click "Generate New Wallet"
3. **IMPORTANT**: Copy and save both:
   - Wallet Address
   - Private Key (keep this SECRET!)
4. Store them somewhere safe (password manager, encrypted file)
5. Click **Continue ‚Üí**

**Option B: Import Existing Wallet**
1. Select "Import Existing Wallet"
2. Enter your Sui wallet address (0x...)
3. Enter your private key
4. Click **Continue ‚Üí**

#### Confirmation (Step 3/3)
1. Review your settings
2. Check estimated earnings
3. Click **‚ú® Start Mining**
4. Done! You're now mining!

### Step 3: Monitor Progress
1. Click the extension icon anytime to see:
   - Connection status (green = mining!)
   - Vectors stored
   - Uptime
   - Earnings
2. Keep your browser open to maximize uptime and earn more!

## Troubleshooting

### "Cannot allocate RAM" error
**Problem**: Not enough available memory  
**Solution**: 
1. Close other applications
2. Reduce RAM commitment in settings
3. Restart browser and try again

### Status shows "Offline"
**Problem**: Can't connect to coordinator  
**Solutions**:
1. Check internet connection
2. Disable VPN if using one
3. Refresh the extension
4. Check [getrem.online/status](https://getrem.online/status)

### Not earning rewards
**Problem**: Configuration issue  
**Check**:
1. Wallet address is correct (in settings)
2. Status is "Online" (green dot)
3. Uptime is above 80%
4. Wait for next epoch (rewards every hour)

### High browser memory usage
**Not a problem!** This is normal - you're providing RAM for mining. 

**If it's too high**:
1. Reduce RAM commitment
2. Close unused browser tabs
3. Restart browser
